About Me

My programming journey began with my fascination of Minecraft. I became obsessed first with building, but once I discovered the the game had its own built in programming system, I became hooked on coding. I bought a Minecraft server and programmed in a custom scripting language called Skript, which was a lot like Python. Since Minecraft was written in Java, I couldn't wait to learn the language. Over the summer, I watched TheNewBostons Java tutorials, and learned basic Java.

During the next few months I started to become frustrated with my progress, since I did not know how to make exciting programs, until I found The Coding Train's p5.js tutorials by Daniel Shiffman. Not only was Daniel Shiffman educational, he was also fun! I proceeded to binge watch The Coding Trainâ€™s entire series in a couple of weeks. Daniel Shiffman's love for programming was infectious, as I was now able to create much more interesting, visual games. After concentrating on javascript for a while, I decided to go back to Java, but this time I used the processing library, which is the Java version of p5.js.

During Spring Break 2018, my parents surprised me by taking me to New York to meet Daniel Shiffman. As a professor at NYU, he gave me a tour of the technology department, including his YouTube studio where I have enjoyed watching and learning his tutorials. With a clear passion for coding, this was an incredible experience, to put it lightly.

I wanted to make more complex games, with an actual game engine, so I watched Brackey's tutorials to learn C#, along with Unity. I participated in multiple game jams on itch.io, which were all awesome experiences. I went back to Javascript, and created a Genetic Algorithm, and eventually a Neural Network, which I would translate into Java to teach my computer to play flappy bird.

Today, I am focused on learning Python since it is one of the most powerful languages for Machine Learning. With certainty, I plan to optimize my high school years learning as much coding and math as possible before entering college as a Computer Science major. Equally, I like sharing my games, programs, coding, resources, and knowledge that helps others learn and enjoy the world of coding. I expect to enjoy a long and productive career in the field of coding and technology. Now let's just see what I create next!
